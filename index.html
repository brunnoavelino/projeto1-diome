O Inicio.

Voltar ao menu

Por mais irônico que pareça ser, tudo começou do inicio. Aprendendo primeiro a montar o HTML em si. O VSCode ajudou bastante basicamente preenchendo uma parte chatinha de gravar, apenas digitando HTML5.
Porém vi que apenas praticando consigo gravar certas coisas, como criar uma lista ordenada usando o < ol >, caso queira, uma lista sem numeração e ordem, mude para < ul >
só não pode esquecer de seus filhos, < li >. Para criar um Título uso o amigo < h1 >, ele vai do maior para o menor, h1, h2 ... por ai vai até o 6.

Desenvolvimento.

Voltar ao menu

Comandos básico para formatação de texto:

  • Itálico: < li >
  • Negrito: < strong >
  • Sublinhado: < u >
  • MarcaTexto: < mark >

Para fazer uso do code, deverá remover os espaços. ^^


Falamos de um monte de formatação ou até mesmo sobre listas, assunto passado anos a frente, mas não poderia esquecer sobre html, head e body.
Até por que sem eles, não disso estaria aparecendo para os senhores. Como vi na aula, para criar um scroll na page, oriento aos senhores que coloque a página em 150%~200% para ficar melhor.
